Featured Project

What I've built

AI-Driven Network & IoT Device Management Using Agentic Automation Workflows
Sep 2025 – May 2026

A four-member undergraduate research project that designed and built a platform integrating automation, network simulation, and intelligent workflow orchestration. Users interact via natural language to automate network diagnostics, device monitoring, incident reporting, and infrastructure management tasks.

Key contributions included developing n8n-based automation workflows, integrating Netmiko, NAPALM, SNMP, and Paramiko for network management, deploying simulated enterprise environments in GNS3, and applying AI-assisted decision-making for operational workflows - all containerized with Docker on a Proxmox hypervisor.
